# Gerald & Alyscia — 15 Valentines

A beautiful, romantic Valentine's Day web experience celebrating 15 years together.

## Features

- Elegant single-page design with gold and wine theme
- Photo gallery slideshow of memories
- Love letter and scripture
- Background romantic music (toggle in bottom-right)
- Responsive and mobile-friendly

## How to View

1. Open `valentines.html` in a web browser, or
2. Run a local server: `python3 -m http.server 8765` and visit http://localhost:8765/valentines.html

## Project Structure

- `valentines.html` — Main page
- `music/` — Background music
- `*.jpg`, `*.png`, `*.gif` — Photos and images
